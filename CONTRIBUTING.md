# Contributing to Human Brain Connectomics Database

Thank you for your interest! 🧠

## Ways to Contribute

### 📊 Data Contributions
- Report outdated or incorrect connectivity data via a [bug report](../../issues/new?template=bug_report.md)
- Suggest new datasets (HCP, OpenNeuro, etc.) via a [feature request](../../issues/new?template=feature_request.md)
- Submit corrections referencing primary sources

### 💻 Code Contributions
1. Fork the repository
2. Create a feature branch
3. Commit with descriptive messages
4. Open a Pull Request
